# Bütçe Takip
Bu proje kullanıcıların gündelik hayatta veya uzun vadeli planlarda bütçelerini takip etmelerini sağlar.

## Özellikler
Proje kullanıcının harcama adedini girmesini sağlar. Ve sırayla harcama adetlerinin tutarının girilmesi sağlanır. Girilen tutarların ortalaması hesaplanarak kullanıcıya bilgilendirilir. Aynı zamanda kullanıcı eğer harcama adedini geçersiz(0) girerse geçerli bir harcama adedi girmesi gerektiği bilgilendirilir.

## Kullanım
İlk adım editörünüzden `main.py` adlı dosyayı açmak. Daha sonra terminale `python main.py` yazarak uygulamayı çalıştırabilirsiniz.
